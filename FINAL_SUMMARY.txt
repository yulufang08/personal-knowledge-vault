================================================================================
                       🚀 快速部署方案
================================================================================

方案1: 本地Docker (开发/演示) - 推荐 ⭐⭐⭐⭐⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
命令:    docker-compose up --build
部署时间: 30秒
访问地址:
  - 前端: http://localhost:3000
  - 后端: http://localhost:3001
  - 数据库: localhost:5432
要求:    Docker + Docker Compose


方案2: Railway (全自动) - 推荐 ⭐⭐⭐⭐⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
步骤:
  1. 访问 https://railway.app
  2. 用GitHub登录
  3. 选择本仓库
  4. 点击"Deploy"
  5. 等待2-5分钟
部署URL:   https://your-app.railway.app
自动功能:  ✅ PostgreSQL数据库
          ✅ 自动HTTPS
          ✅ GitHub自动更新
成本:      免费 ~ $7/月


方案3: Vercel + Railway (混合) - 推荐 ⭐⭐⭐⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
前端 (Vercel):
  1. https://vercel.com 导入项目
  2. 选择 /frontend 目录
  3. 部署

后端 (Railway):
  按照方案2部署

优点:     国内访问快, 全球CDN加速
成本:     免费 ~ $15/月


方案4: Render (一体化) ⭐⭐⭐
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
步骤:
  1. https://render.com
  2. 选择 "New Web Service"
  3. 连接GitHub仓库
  4. 配置构建命令
  5. Deploy

配置:
  - Build: npm install && npm run build
  - Start: npm start
成本:     免费 ~ $12/月


方案5: 自托管 (完全控制)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
平台:    AWS EC2, 阿里云, 腾讯云, DigitalOcean
优点:    完全控制, 中国服务器
成本:    $5~50/月
难度:    需要服务器运维经验

                      🔌 API端点一览
================================================================================

笔记操作:
  GET    /api/notes              - 获取笔记列表
  POST   /api/notes              - 创建笔记
  GET    /api/notes/:id          - 获取笔记详情
  PUT    /api/notes/:id          - 更新笔记
  DELETE /api/notes/:id          - 删除笔记

标签操作:
  GET    /api/tags               - 获取所有标签
  POST   /api/tags               - 创建标签

搜索与关系:
  GET    /api/search             - 全文搜索
  GET    /api/graph              - 知识图谱数据

系统:
  GET    /api/health             - 健康检查

                      💬 常见问题
================================================================================

Q: 需要支付费用吗?
A: 不需要! Railway、Vercel、Render都提供免费额度,足以运行此应用。

Q: 部署需要多长时间?
A: 5-10分钟。下载依赖和构建镜像大部分时间。

Q: 可以自己搭建服务器吗?
A: 可以! 项目已配置Docker,支持任何云平台部署。

Q: 数据存储在哪里?
A: PostgreSQL数据库存储在云平台(Railway/Render等)。

Q: 支持移动访问吗?
A: 完全支持! 前端已响应式设计,手机上体验完整。

Q: 可以离线使用吗?
A: 当前版本需要网络。V2.0计划支持PWA离线模式。

Q: 如何备份数据?
A: 支持导出为JSON格式。云平台也提供自动备份。

Q: 支持多用户吗?
A: 当前版本针对个人使用。企业多用户功能在V3.0计划中。

Q: 遇到问题怎么办?
A: 查看docs目录的故障排除指南,或查看日志。
